# Oculina_sc_manuscript

This repository contains the code for the manuscript: Cell type-specific immune regulation under symbiosis in a facultatively symbiotic coral" by Maria Valadez-Ingersoll et al (2025): https://doi.org/10.1093/ismejo/wraf132. 
Within this repository are folders for the single-cell analysis, the 16S analysis, and the physiology analysis. The proteomics analysis was performed by Jeric Da-Anoy and can be found here: https://github.com/jpdaanoy2024/Proj_O.arbuscula_proteomics/tree/main
